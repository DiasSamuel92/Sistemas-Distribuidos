# Calculadora RMI

## Portas:
- Porta do webservice REST: 8081
- Porta do Servidor RMI Básico: 10999
- Porta do Servidor RMI Avançado: 10998

## Rotas:
- /somar
- /subtrair
- /dividir
- /multiplicar
- /porcentagem
- /raiz
- /potencia

Os dois numeros devem ser enviados nos parametros: num1 e num2.
